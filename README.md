Custom Contact Form

Custom Contact Form is a lightweight and user-friendly WordPress plugin that allows you to add a simple and secure contact form to your website. It is ideal for small websites, portfolios, and business sites that need a clean contact solution without extra complexity.

Features
Simple contact form (Name, Email, Subject, Message)
Secure and lightweight
Email notification on form submission
Basic form validation
Easy integration using shortcode
Works with any WordPress theme

Installation
Download or clone this repository
Upload the custom-contact-form folder to /wp-content/plugins/
Activate the plugin from the WordPress Admin Dashboard
Add the shortcode to any page or post

Usage
Use the following shortcode to display the contact form:
[custom_contact_form]

Requirements
WordPress 5.0 or higher
PHP 7.0 or higher
